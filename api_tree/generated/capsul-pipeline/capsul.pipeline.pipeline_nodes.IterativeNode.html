capsul.pipeline.pipeline_nodes.IterativeNode¶

class capsul.pipeline.pipeline_nodes.IterativeNode(pipeline, name, process, iterative_plugs, do_not_export, make_optional, **kwargs)[source]¶

A special node to store an iterative process.

This node is dynamic and try to be updated when a trait value is modified.
